Dokumentacja

Projekt dotyczy światła jak i zjawiska jego rozszczepienia.

W ramach języka projektu wykorzystano język HTML5 wraz z CSS i Java Script.

Na stronie mamy możliwość odtwarzania muzyki z elementu audio. Odtwarzana jest ona automatycznie (metoda onload). Dodatkowo mamy możliwość jej pauzowania i wznawiania, za co odpowiedzialna jest metoda onclick, która dodatkowo zmienia ikonę za pomocą atrybutu CSS (style.display).

Element canvas zawiera prostą animację wykonaną w JavaScript'cie. Używa ona pomocniczej funkcji "sleep" aby rysowanie obiektów było widoczne w czasie.
